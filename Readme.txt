选择http://www.benmuji.cn/news/20220612412.html中列出的top10的数字加密货币
用户初始时，用户名为New User，初始持有一万美元，十万人民币。

该App共有三个页面：市场页，交易页，个人页。

个人页：
显示用户名，用户头像（可以点击选择其他头像），当前持有哪些数字货币和货币

交易页：
展示所有支持的10种数字加密货币到指定的法定货币之间的汇率关系。
在刚切换到交易页面时，会拉取所有汇率关系；在拉取到汇率之前，显示汇率处会以"..."做占位符，且此时尝试点击展开购买页，会提示请等待网络传输。
在汇率拉取完毕后，再点击任意加密货币，会展开新的页面，可以购买该货币。
购买页允许用户输入要购买的数字货币数量，同时会显示用户当前持有的法定货币数、购买需要消耗的法定货币数、购买后剩余的法定货币数。
如果购买数为不合法的输入值、负数、或购买所需的法定货币超过所持有的所有法定货币，会toast提示相关错误，禁止购买。
用户成功确认购买后，会提示购买成功，并关闭购买页。此时再切换到个人页，会看到新买的货币，以及消耗了一些法定货币。

市场页：
显示最近的交易历史价格（方便用户做参考决策
